# Weather App

Une application météo simple et interactive développée avec React, permettant de rechercher la météo en temps réel pour n'importe quelle ville.

## Fonctionnalités

- Recherche de la météo par ville

- Affichage des conditions météo actuelles (température, humidité, vitesse du vent)

- Icônes dynamiques en fonction des conditions météorologiques

- Interface utilisateur moderne et responsive

## Technologies Utilisées

- React.js - Framework JavaScript pour l'interface utilisateur

- CSS - Pour le style et la mise en page

- OpenWeather API - Pour récupérer les données météo en temps réel

## Installation et Exécution

1.  Clonez le repository :

    git clone https://github.com/votre-utilisateur/weather-app.git
    cd weather-app

2.  Installez les dépendances :

    npm install

3. Démarrez l'application :

    npm start

4. Ouvrez http://localhost:3000 dans votre navigateur.
